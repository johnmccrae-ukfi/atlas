# Atlas Research

This directory contains datasets and research artefacts used to develop and validate Atlas analytical models.

## Purpose

Research datasets are not part of the operational ingestion pipeline.

They are used for:

- feature engineering
- backtesting
- AI experimentation
- model validation
- historical analysis

## Dataset Categories

- legacy
- purchased
- generated
- sample

Operational data should never be stored here.

Operational Bronze/Silver/Gold data belongs in Microsoft Fabric.
